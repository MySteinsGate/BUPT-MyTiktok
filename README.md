## Feature

- 使用 RecyclerView 显示视频列表，一页显示多个 item ✔ 

- 使用 Glide 加载封面图 ✔

-  显示每个视频必要的信息 ✔ 

-  采用 ViewPager2 来实现一页显示一个 item ✔ 
- 从视频信息流点击某个视频封面进入播放页面 ✔ 
- 根据视频信息的 URL 播放视频 ✔ 
- 视频加载过程中，播放 Loading 动画 ✔ 
- 单击视频窗口暂停/继续 ✔ 
- 双击视频窗口弹出点赞爱心图标 ✔
